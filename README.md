# Smart Resume Skill Gap Analyzer

An AI-powered web application that analyzes resumes and identifies skill gaps for various job roles across Software, Data Science, and Electronics domains.

## Features
- PDF resume upload and text extraction
- Skill gap analysis for multiple job roles
- Visual analytics (pie chart, bar chart)
- Electronics, Software, and Data Science roles supported
- Clean, modern Streamlit UI

## Tech Stack
- Python
- Streamlit
- Matplotlib
- PyPDF2

## How It Works
1. Upload your resume (PDF) or paste text
2. Select a target job role
3. Get skill match percentage and missing skills
4. Visualize skill distribution

## Installation & Run
```bash
pip install -r requirements.txt
streamlit run app.py
```
